# Costco Reciept Analyizer
Understand your spending at Costco

## Description 
This html was copied from a [post on reddit](https://www.reddit.com/r/Costco/comments/1p4w903/) That I found quite interesting. 

Script that automates receipt download: https://github.com/ankurdave/beancount_import_sources/blob/main/download/download_costco_receipts.js

##  Step by Step Instructions:
- Login to costco.com
- Go to https://www.costco.com/OrderStatusCmd.
- Paste the script from the ankurdave/beancount_import_sources repo linked above into your browsers developer console.
- ⁠Open html in browser
- ⁠Open the `receipts.json` file downloaded by the scraper in the html page
- ⁠Your personal Costco Dashboard is now ready.
